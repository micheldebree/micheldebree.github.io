Iceball, a tune made in assembler code

January 1, 1996 15-minute read
music • c64

A tune made in assembler code

This tune was made on the Commodore 64 home computer somewhere in 1996. Although there were plenty of music editors to create music in by that time, I chose a very arcane way of doing things; by using assembler code.

The reason to do it like this is because I was after the specific sound of the excellent music routine by Laxity, and also to learn about how such a music routine works internally. To do this, I reversed engineered the binary data of one of Laxity’s tunes. The source code of the tune is below the video.

Source code
